和自己许一个五年之约

发表于 2017-03-28 | 分类于 我有一个徒步中国的梦 | | 阅读次数 |

过年在家整理东西的时候，偶然发现了高中时候买的一本书《十八岁之前禁止涉足的十八个地方》，这本书可以说是我的一个梦的引子。它让我在紧张的备考期间不断幻想那种生活。

大学这几年里，自己也走过一些地方。每次走的履历不同，心情不同，很遗憾地是没有记录下来。所以从这篇文章开始，用枯燥的文字去记录那多彩的经历，当累的时候去看一看，告诫自己心中的梦。

现如今，距我的十八岁已过去五年，再次去回想当初的那些想法，说真的，会有些愧疚，对不起当时的自己。当时想的很多，如今做的很少。
不管怎样，最近一直在想要换一个生活方式，若真的要去描述这个生活方式是什么，我恐怕真的不知道，但只是有一点很清楚，它是健康的，无论是身还是心。
敢想更要敢做。
所以和自己不妨来场五年之约，在未来的五年里，走完这本书里未涉及的地方，并用文字记录下来，更是用行动给自己的青春的一篇文章画上一个句号。








附上自己的当时随书笔记（也算青春期的文艺气息八）

《一》
梦在远方
路在脚下

《二》
丽江
说实话，没有什么可吸引我的
如果非要找的话
希望有朝一日在荡在河里的小船上
安安稳稳地睡一觉
醒来时
已是黄昏

《三》
山是水的首
水是山的肩
相互陪伴
一叶竹筏
一曲民谣
飘在水间
游在山间
虽没亲临
闭上眼
便能体验


《四》
水的尽头是山
山的尽头是水
世间竟有如此清澈的水
一朵朵云溶在水中


《五》
旧路新走不扰仙
雪山古寺藏云间

《六》
人们说天地是相连的
如果云是天派来的使者
那么高高的青杨
则是大地热情欢迎时伸出的手

《七》
一瓣，两瓣，三瓣，四瓣，五瓣
由浅入深
有聚到绽
在释放自己的同时
也诠释了生命


《八》
马儿
你苦么
每天是新的一天
却重复着旧路
在的虽是不同的人儿
可他们的说笑与你无关

愿你我在追梦路上相互搀扶！
